## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Running a daycare is never easy, you have kids,parents, staff, investors and regulatory requirtements you have to juggle. The Safari Life CMS is intended to ease some of the administrative workload on child care/pre-k educators and administrators to allow them to get back to their real work: Shaping Young Minds!
